### 1 - What is Reinforcement Learning (RL)?

Reinforcement Learning is a way of teaching an **agent** through **trial and error**.  
If the agent takes a good action, it receives a **reward**. If it takes a wrong action, it receives a **penalty** (negative reward).

A simple example is how a baby learns to walk:  
At first, they fall many times, but with repeated attempts, they learn how to stand and eventually walk on their own.
